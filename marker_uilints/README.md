# Marker's UI tests

UI tests for Marker, implemented as a lint crate.

This is also a playground to experiment with markers API.

## Usage

```sh
# Running these UI tests
cargo uilints

# Blessing the `.stderr`, `.stdout`, `.fixed` files
cargo uilints -- -- --bless
```
